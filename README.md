# 📧 Email Automation using Google Sheets

## 🔹 Description

This project automates sending emails using Google Sheets and Google Apps Script.

## 🔹 Features

* Send emails automatically
* Bulk email support
* Personalized messages
* Button-based trigger

## 🔹 Technologies Used

* Google Sheets
* Google Apps Script
* GitHub

## 🔹 How it Works

1. Add email & name in sheet
2. Click "Send Emails" button
3. Emails are sent automatically
4. Status updates to "Sent"
